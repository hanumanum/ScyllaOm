# ScyllaOm
ScyllaOm is a simple ORM for ScyllaDB (& Apache Cassandra). It is a simple wrapper around the [DataStax Node.js driver]

## Instalation
```
npm install git+ssh@github.com:hanumanum/ScyllaOm.git
```
